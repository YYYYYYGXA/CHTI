# 吃货TI - 本命食材MBTI测试

一个有趣的MBTI风格人格测试网站，将你的性格与本命食材联系起来，还可以邀请朋友一起"做菜"！

🍳 在线体验：https://yyyyyyygxa.github.io/CHTI

## 功能特点

- 📊 12道趣味MBTI风格测试题
- 🥕 12种专属本命食材
- 👥 多人互动，组合食材做料理
- 🎨 精美的渐变UI设计

## 技术栈

- React 18
- Vite 5
- 纯CSS动画

## 本地运行

```bash
npm install
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 部署

项目已配置GitHub Actions，自动部署到GitHub Pages。
